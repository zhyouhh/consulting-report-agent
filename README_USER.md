# 咨询报告写作助手 - 使用说明

## 简介

这是一个帮助快速生成专业咨询报告的智能助手工具。

## 系统要求

- Windows 10/11
- 无需安装其他软件，双击即可使用

## 首次使用

1. 双击 `咨询报告助手.exe` 启动程序
2. 程序会自动打开窗口，显示项目列表

## 创建报告

### 1. 新建项目

点击左侧"新建项目"按钮，填写：
- **项目名称**：给报告起个名字
- **报告类型**：选择以下之一
  - 专题研究报告
  - 体系规划方案
  - 实施方案
  - 管理制度
- **主题**：报告的核心内容
- **目标读者**：报告面向的人群

### 2. 设计大纲

创建项目后，助手会引导你：
1. 确认报告主题和目标
2. 设计报告大纲结构
3. 确认各章节内容要点

### 3. 撰写内容

大纲确认后，助手会：
1. 逐章节撰写内容
2. 主动询问关键信息
3. 确保数据真实性
4. 避免使用AI化词汇

### 4. 质量审查

内容完成后，助手会：
1. 检查逻辑连贯性
2. 验证数据来源
3. 确认格式规范

### 5. 导出报告

审查通过后，可以导出为：
- Markdown 格式
- Word 文档（需要转换工具）

## 注意事项

- 所有数据必须真实，严禁编造
- 数据需标注来源
- 避免使用"赋能、抓手、闭环"等词汇
- 报告内容会自动保存在用户目录

## 数据存储

- 配置文件：`C:\Users\你的用户名\.consulting-report\config.json`
- 项目文件：`C:\Users\你的用户名\.consulting-report\projects\`

## 常见问题

**Q: 程序无法启动？**
A: 检查是否被杀毒软件拦截，添加信任即可。

**Q: 如何修改 API 配置？**
A: 编辑配置文件 `~\.consulting-report\config.json`

**Q: 报告保存在哪里？**
A: 在 `~\.consulting-report\projects\项目名称\` 目录下
