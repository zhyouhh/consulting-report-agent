# 咨询报告写作助手 - 使用说明

## 简介

这是一个面向内部同事的 Windows 桌面客户端，用来推进咨询报告写作。

## 系统要求

- Windows 10/11
- 解压后直接双击即可使用

## 首次使用

1. 双击 `咨询报告助手.exe`
2. 程序会自动打开桌面窗口
3. 默认显示 `默认通道`
4. 如果默认通道可用，可以直接开始写

## 连接模式

### 默认通道

- 推荐模式
- 开箱即用
- 默认模型：`gemini-3-flash`

### 自定义 API

- 高级模式
- 适合已经有自己 API 的同事
- 可在设置里自行填写地址、密钥和模型

## 创建项目

点击左侧 `新建报告` 后，填写：

- 项目名称
- 项目类型
- 报告主题
- 截止日期
- 预期篇幅
- 目标读者
- 备注

## 工作流程

1. 新建项目
2. 在中间聊天区和助手讨论需求
3. 在右侧工作区查看当前阶段和项目文件
4. 需要时运行质量检查
5. 导出 `可审草稿`

## 当前导出说明

- 当前支持导出 `可审草稿`
- 这是给内部审阅和继续修改用的
- 不是最终定稿排版版本

## 数据存储

- 配置文件：`C:\Users\你的用户名\.consulting-report\config.json`
- 项目文件：`C:\Users\你的用户名\.consulting-report\projects\`

## 常见问题

**Q: 默认通道不能用怎么办？**  
A: 先打开设置，确认仍是 `默认通道`；如果默认通道暂时异常，可以切到 `自定义 API`。

**Q: 程序无法启动？**  
A: 先检查是否被杀毒软件拦截，必要时添加信任。

**Q: 项目文件保存在哪里？**  
A: 在 `C:\Users\你的用户名\.consulting-report\projects\项目名称\`。
